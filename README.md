# ⚡ ResumeAI — The Ultimate Resume Builder

> Handcrafted by **Arun Pratap Singh** (b24bs2044@iitj.ac.in) for the **Digital Heroes** developer trial.

## 🚀 What's the hype?

I got tired of boring, clunky resume builders, so I built one that actually feels *insanely good* to use. 

This isn't your average forms-and-submit app. I engineered a **premium, glassmorphism-inspired dark mode editor** that runs blazing fast on React 19 + Vite 7. You type on the left, and your pixel-perfect A4 resume updates instantly on the right. No lag, no page reloads, just pure aesthetic development. 

Whether you're a designer wanting that elite 'Digital Heroes' vibe or a backend dev who just needs an ATS-friendly layout fast, this app has you covered.

## 🔥 Badass Features

- **4 Next-Level Templates**: Classic, Modern, Minimal, and Minimal+Image. Built to beat ATS while looking gorgeous.
- **Real-Time Engine**: Type your skills, see them instantly render. We even have custom zoom controls (40% - 120%) so you can inspect every pixel.
- **Theming System**: Pick from 6 curated accent colors or dial in your own custom hex code to match your personal brand.
- **Zero-Friction Storage**: No login required to start hacking your resume. Everything saves directly to your browser's local storage.
- **Flawless Export**: Clean A4 PDF generation. The print stylesheet completely strips the dark mode UI and spits out a crisp, white, print-ready document.
- **Mobile Responsive**: Hacking from your phone? Just hit the toggle button at the bottom to switch between the editor and the preview canvas.

## 🛠 Tech Stack (The Good Stuff)

- **Frontend Core**: React 19 + Vite 7 (Because speed is everything)
- **Styling Engine**: Tailwind CSS 4 (Custom animations, glassmorphism, and dynamic gradients)
- **Icons**: Lucide React
- **Architecture**: No-backend local-first approach

## 📦 Spin It Up

Want to run it locally? It takes 10 seconds.

```bash
# Install dependencies
npm install

# Fire up the dev server
npm run dev

# Build for production
npm run build
```
